# 模組 2：內質 `inner_nature`

## 核心責任

建立角色不可見但持續驅動行為的心理架構：價值、欲望、缺陷、恐懼、衝突、情緒、依戀、決策與防禦機制。

## `data` 固定格式

```yaml
人物核心定義: {}
基礎組成: {}
內在驅動力: {}
情感架構: {}
潛意識與精神衝突: {}
行為偏好與習慣: {}
過去經歷: {}
極端情緒: {}
人際網絡與歸屬感: {}
內質的生動表達: {}
```

各區塊內繼續展開語意明確的子鍵，例如 `基礎組成.基礎認識`、`內在驅動力.精神核心`、`情感架構.情緒運作`、`潛意識與精神衝突.內心衝突`。不得把這些欄位改寫成 Markdown 粗體段落。

## `data` 填寫指引

### `core-definition` 人物核心定義

- 核心特質、核心缺陷、自我定位與隱藏的人格定位。
- 心理類型只能作為一致性工具；不可用 MBTI 或九型標籤取代行為機制。

### `foundation` 基礎組成

- 稱謂與自我認知、心智成熟度、性別認同、社會角色。
- 淺層與深層性格標籤、主要性格、次要特質與心靈意象。

### `drives-and-values` 內在驅動力

- 對世界的態度、人生追求、價值取向、道德底線、生命意義與禁忌。
- 思維模式、決策風格、風險承受、壓力應對與逃避方式。
- 3 至 5 個核心訴求、實現路徑與當下追求。

### `emotional-architecture` 情感架構

- 情緒運作、表達方式、依戀類型與情感變化規律。
- 愛情觀、親密需求、社會情感、親情、友情與陌生人信任。
- 成人親密觀念只在 Blueprint 要求時寫，並交代其人格功能而非單純列玩法。

### `subconscious-conflict` 潛意識與精神衝突

- 潛在追求、恐懼、心理弱點、信念衝突、自我矛盾與隱藏創傷。
- 說明它們如何滲入日常決策，而非只列悲慘事件。

### `habits-and-coping` 行為偏好與處事哲學

- 生活習慣、興趣、喜惡、自我評價、待人方式。
- 衝突應對、自我保護、失敗反應與修復方式。

### `extremes-and-belonging` 極端情緒與歸屬

- 人生支柱、依賴事物、不可失去的存在。
- 爆發條件、極端情緒種類、承受方式、社交與存在感需求。

### `living-expression` 內質的生動表達

- 扮演核心、內心獨白。
- 友好、敵對、親密等狀態的短台詞。
- 3 至 5 個能顯示心理機制的場景，但不要重複模組 6 的長篇語料。

## 品質檢查

- 缺陷必須真實影響選擇，不是可愛裝飾。
- 每個極端狀態都有具體觸發與過渡。
- 心理類型、價值觀與實際行為互相吻合。
