# 模組 3：外延 `extension`

## 核心責任

把角色錨定在社會與世界中：背景、成長、身分、能力、職業、階層、生活方式、人際網絡與環境差異。

## `data` 固定格式

```yaml
人物核心定義: {}
背景設定與成長經歷: {}
人際關係: {}
社會階層與經濟狀況: {}
行為模式: {}
情緒表現特徵: {}
生活習慣與個性特徵: {}
外延的生動表達: {}
對 {{user}}: {}
性相關: {}
人物意象: {}
私人空間營造: {}
差異表現補充: {}
```

## `data` 填寫指引

### `social-identity` 社會身分

- 社會稱謂、年齡與生日、身分、所在地、經濟狀況、所屬團體與簡介。
- 5 至 8 個外延標籤，以及職業、天賦、技能、興趣、喜惡與嗜好。

### `background-and-growth` 背景與成長

- 家庭成員、家庭背景、家庭態度。
- 出生、童年、教育、成長軌跡與重要人物。
- 只選真正塑造當前角色的關鍵經歷，避免流水帳。

### `social-network` 人際關係

- 社交需求、社交圈、依賴程度、需求與拒絕傾向。
- 當前朋友、團體、服務／使役關係與特殊感情對象。
- 初見他人的評估方式、信任形成、容易好感或厭惡的特質。

### `class-and-function` 階層、經濟與社會職能

- 社會階層、外界評價、自我看法。
- 權利、責任、義務、盟友、敵對與潛在關係。
- 階層與職業必須影響詞彙、習慣、資源與風險選擇。

### `social-behavior` 社會行為模式

- 社交傾向、社交語言、形象塑造與環境差異。
- 決策邏輯、問題解決、權力與控制欲、道德與利他性。
- 壓力、失敗、暴力、服從、信任與忠誠的社會表現。

### `daily-life` 生活習慣

- 作息、飲食、衛生、空間整理與特殊習慣。
- 好感對象前的心理、行為與追求方式。

### `roleplay-anchor` 外延的生動表達

- 扮演核心、3 至 5 個扮演要點與誤區。
- 3 至 5 個代表性社會場景，展示不同權力與環境下的行為差異。

### `user-relationship` 對 `{{user}}`

- `初始看法`：初始關係、感情態度、認知、在意程度及 5 至 8 種互動模式。
- `人物經歷`：過去經歷、重要事件、角色自己的看法及可選期望關係。
- `行為細化`：不同情境的稱呼、說話模式與肢體互動。
- `情感變化路徑`：分別寫交往前、交往後、發生性關係後的行為、心理與語氣。

### `sexual-and-private` 性相關、意象與私人空間

- `性相關` 分為性經驗、性知識、性習慣；性習慣需含 4 至 7 項行為習慣、4 至 7 項偏好、3 至 5 項擅長技術及發情前後、前戲、過程、事後反應。
- `人物意象` 固定為專名、氛圍意象、物質意象。
- `私人空間營造` 寫空間類型、風格、佈置與角色在其中的行為。
- `差異表現補充` 的環境性、身份性、氛圍性表現各 3 至 5 項。

## 品質檢查

- 背景事件能解釋但不機械決定現在的性格。
- 職業與階層有可見後果，不只是標籤。
- 舊 `expanded_extension` 的全部責任已合併到本模組，不得另產生第 4 份外延擴展。
