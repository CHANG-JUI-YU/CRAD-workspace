# Greetings Generation Guide

每個 greeting 建立可立即互動的時間、地點、在場角色與張力，以感官或角色行動提供切入點，將玩家反應保持開放。每則以約 350 至 600 個中文字、至少三個自然段落為軟目標，不是必須湊滿的 schema 字數。

完整場景依序涵蓋：場景落地與感官線索、角色登場及符合設定的聲線或行動、具體事件或衝突、張力或關係狀態的推進，以及玩家可立即回應的互動鉤子。可以描寫角色與環境，但不得替玩家宣告思想、感受、對白或既成行動。禁止同義反覆、形容詞堆疊及只為增加篇幅的空泛內容。

Alternate 應提供實質不同的事件、關係張力或互動局面，不只是更換地點或措辭；group-only 明確記錄適用群像，不混入單角色 primary。
