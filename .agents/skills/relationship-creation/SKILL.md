# Relationship Creation

依 `references/generation-guide.md` 與 Task-bound exact artifacts 建立唯一 `relationships.yaml` replacement proposal。只分析設定，不扮演角色，不輸出私密 planning 或 thinking。

輸出只能是 `proposal@1` 的 `relationships` value；不得直接寫作者檔、產生編譯 team 標記或提交其他 kind。
