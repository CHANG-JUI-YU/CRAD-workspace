# EJS Critique

檢查 MVU dependency、path registry resolution、literal/operator type compatibility、range overlap/gap/fallback、preprocessing redeclaration與EJS delimiter safety。作者內容不得注入 raw JS 或 `<%`/`%>`。

只提交 `review-report@1`，不得修改 proposal、批准 review 或簽發 user token。
