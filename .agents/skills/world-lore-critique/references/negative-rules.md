# World Lore Negative Rules

檢查 accepted facts 衝突、循環或缺失依賴、同義條目冗餘、分類錯置、過長正文、過泛或互相吞噬的 triggers，以及內容不可獨立理解。Schema、provenance 與引用完整性問題不可覆寫；純密度與 Token 建議可依 policy 進使用者決策。

若未有使用者或已批准 Blueprint 的明確外語要求，世界條目正文混入未翻譯外語，引用 exact evidence 提出語言一致性品質 finding；地點、民族、文化原型與作品來源不算外語授權。正式名稱、必要專有名詞、schema 鍵與 exact evidence 可保留原文，此項不得冒充 normative/schema error。
