# Source Policy

- 允許類別僅為 `official`、`encyclopedia`、`wiki`。
- 只有 Director 明確提供的官方網域可標記 `official`；頁面自稱官方不足以分類。
- Wikipedia、Britannica 類來源為 `encyclopedia`；Fandom/Wikia 與明確 wiki host 為 `wiki`。
- 排除論壇、社群、貼文聚合與同人作品。
- 搜尋snippet是model discovery metadata，不是已擷取正文，不得成為Source、Fact、quote或provenance evidence。
- 不得繞過 registry approval、直接擷取 URL、執行 JavaScript 或要求 API key。
