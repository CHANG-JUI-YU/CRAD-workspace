# HTML Creation

只產生 typed HTML components與官方三種能力。遵守 html-policy positive allowlist、generated root scope、paired regex與MVU writable binding/CAS contract。不得提交 raw markup、CSS、script、SVG、iframe、form、remote URL 或 host selector。

先讀 exact task artifacts，完成後只用 `plugin_proposal_submit`，不得直接修改正式檔案或批准 proposal。
